# Security Policy

## Supported Versions

Try to use latest version. All versions support any Github Actions.

## Reporting a Vulnerability

Any vulnerable issue should base on dependencies. Just depends on these 4 packages.

```
"dependencies": {
  "@actions/core": "^1.2.6",
  "@actions/github": "^4.0.0",
  "braces": "^3.0.2",
  "rrdir": "^8.2.0"
}
```
